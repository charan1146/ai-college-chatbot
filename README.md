# 🎓 AI College FAQ Chatbot

A beginner-friendly AI chatbot developed using Python and Streamlit.

## Features
- Answers college-related questions
- Simple keyword-based NLP
- User-friendly interface
- Handles queries about:
  - Exams
  - Fees
  - Hostel
  - Library
  - Holidays
  - Canteen

## Technologies Used
- Python
- Streamlit

## Future Improvements
- Voice assistant
- Database support
- ChatGPT API integration
- Multi-language support
